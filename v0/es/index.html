    Table of Contents¶

    vrpRouting extends the PostGIS/PostgreSQL geospatial database to provide geospatial routing and other network analysis functionality.

    This is the manual for vrpRouting v0.4.1.

Feel free to use this material any way you like, but we ask that you attribute credit to the vrpRouting Project and wherever possible, a link back to https://pgrouting.org.
